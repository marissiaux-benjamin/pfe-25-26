# PFE 2025-2026
## Plateforme E-commerce pour une Artisane

---

## Présentation du projet

Ce Projet de Fin d'Études consiste en la conception et le développement d’une plateforme e-commerce dédiée à une artisane spécialisée dans :

- Vitraux artisanaux
- Galets
- Lampes en vitrail

L’objectif est de créer une plateforme :

- Intuitive
- Simple d’utilisation
- Moderne
- Adaptée aux besoins réels de l’artisane

La solution comprendra :

- Une partie client
- Une partie administration

Le projet sera développé avec :

- Laravel
- Livewire
- MySQL
- Blade
- Tailwind CSS (ou autre framework CSS)

---

# Objectifs du projet

## Objectifs fonctionnels

- Présenter l’univers et le savoir-faire de l’artisane
- Permettre l’achat en ligne de produits
- Gérer le stock et les commandes
- Proposer un formulaire de contact
- Fournir un espace administrateur complet

## Objectifs techniques

- Implémenter une architecture MVC propre
- Utiliser Laravel Livewire pour une expérience dynamique
- Assurer la sécurité des données
- Concevoir une base de données optimisée
- Mettre en place un système d’authentification sécurisé
